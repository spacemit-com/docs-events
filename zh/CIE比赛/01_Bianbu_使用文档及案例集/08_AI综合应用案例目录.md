# 1. AI 聊天机器人

**项目意义：**
本案例旨在构建一个本地化的 AI 聊天机器人，集成自动语音识别（ASR）、大语言模型（LLM）以及文本转语音（TTS）等核心技术模块，帮助用户深入理解多模型协同工作的机制与运行原理。

**功能简介：**
用户通过终端启动应用后，将看到语音输入提示。系统会实时录入语音并进行语音识别，随后通过本地部署的 Deepseek R1 模型进行推理与语义理解，最终将生成的文本结果打印在终端中。此外，案例还提供支持 TTS 播报的大模型输出内容的版本，用户可根据实际需求选择是否启用语音合成功能。

# 2. 人脸识别应用

**项目意义：**
通过本地计算机视觉（CV）模型的人脸识别应用案例，帮助开发者理解视频流处理流程，并掌握人脸识别相关应用的开发方法。

**功能简介：**
用户在终端启动演示程序后，系统将启用“人脸识别”功能，实时分析摄像头画面，并与本地人脸图像数据库进行比对。当检测到匹配的人脸时，系统会在画面中框选目标并标注其对应姓名。该应用还支持通过系统相机拍摄照片并存储为新的识别样本，用于后续比对与识别训练。

# 3. 语音控制电机

**项目意义：**
探索人工智能与嵌入式系统的融合应用，基于 Function Call 机制，将语音识别结果精准映射为可执行的物理控制指令，构建典型的 AIoT（人工智能物联网）应用原型。

**功能简介：**
用户在终端启动程序后，系统通过自动语音识别（ASR）与模糊语义匹配算法，识别用户语音指令，并将其转换为控制信号，驱动执行单元（如舵机）完成相应动作，形成语音控制硬件的完整闭环。

# 4. **数字识别系统**

**项目意义：**
构建计算机视觉（CV）与嵌入式系统融合的开发框架，验证CV模型在边缘设备上的部署能力与端侧识别效果，搭建模型部署与物理反馈的原型平台。

**功能简介：**
用户启动终端程序后，系统调用 USB 摄像头实时采集视频流，部署并运行基于 PaddleOCR 的视觉识别模型，对画面中的纸面数字内容进行检测与识别。识别出的数字将被转换为控制信号，驱动数码管进行同步显示，实现从感知到控制的闭环演示。
